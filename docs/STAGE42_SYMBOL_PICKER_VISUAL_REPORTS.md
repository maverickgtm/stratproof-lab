# Stage 42 — Symbol Picker + Visual Evidence Reports

Stage 42 improves the local workbench before GitHub release.

## Added

- Top-symbol checkbox selector for common crypto pairs.
- Custom/rare symbol input for pairs not listed in the default selector.
- Visual Evidence Report HTML output generated alongside Markdown and JSON reports.
- Workbench link: `Open visual evidence report`.
- Visual report cards for known outcomes, honest winrate, Net R, configuration, symbol results, evidence guidance, and safety.

## Symbol selector

Default top symbols:

- BTCUSDT
- ETHUSDT
- SOLUSDT
- BNBUSDT
- XRPUSDT
- DOGEUSDT
- ADAUSDT
- AVAXUSDT
- LINKUSDT
- HYPEUSDT

Rare/custom symbols can still be typed manually, separated by commas.

## Safety

The report remains audit-only. It does not execute trades, send alerts, place broker orders, manage funds, or provide financial advice.
