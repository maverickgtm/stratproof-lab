## Intended README story

1. Hero: “Most trading strategies look good — until they are audited.”
2. Formula Builder: show how users create a hypothesis.
3. Evidence Report: show how StratProof turns results into evidence.
4. Research Brain View: show the system’s audit pipeline visually.
5. Provider Connector Layer: show crypto-first and multi-market future.

## Visual principle

The operational dashboard must remain practical. The Research Brain View exists for visual observability and product storytelling: it shows how the system works, not to replace the audit workflow.
