## Section: The problem

Pretty backtests create false confidence.

A strategy can look profitable because of duplicate entries, leakage, small samples, cherry-picked sessions, ignored fees, regime bias, or same-candle assumptions.

StratProof Lab exists to challenge the result before the market does.

## Section: How it works

1. Build the idea visually.
2. Import or download market data.
3. Run an audit-only backtest.
4. Apply Quality Gates.
5. Compare thresholds, sessions, symbols, and regimes.
6. Generate an evidence report.

## Section: Core screens

### Formula Builder

Create research hypotheses with blocks like RSI, EMA, volume, sessions, score thresholds, and BTC context.

### Evidence Report Builder

Turn audit results into verdict cards: honest winrate, Net R, drawdown, duplicate risk, leakage risk, and evidence score.

### Research Brain View

A visual observability screen showing how the Research University routes ideas through departments and quality gates.

### Provider Connector Layer

Start with CSV and public crypto data. Expand toward multi-market data sources.

## Section: Why it is different

StratProof Lab is not execution-first.

It is evidence-first.

Most tools help you build, optimize, or run strategies. StratProof Lab helps you question whether the strategy should be trusted at all.

## Section: Safety

Audit-only by design. Evidence before action. No broker execution in Community. No guaranteed returns. No financial advice.
