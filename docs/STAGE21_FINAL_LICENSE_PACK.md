# Stage 21 — Final License Pack + Legal Boundary Cleanup

Stage 21 converts the public Community package from "license planned" to "license structure declared".

## Selected license

Community Edition uses:

```text
SPDX-License-Identifier: AGPL-3.0-or-later
```

## Why this license

StratProof Lab is designed to become useful as a local and hosted audit system. AGPL-3.0-or-later protects the public version better than a permissive license because modified hosted versions may have source-availability obligations.

## Publication blocker still remaining

Before public GitHub release, replace `LICENSE` with the full canonical GNU AGPL v3 license text copied directly from the Free Software Foundation or generated by GitHub's AGPL-3.0 license template. Do not hand-edit the license text.

Reason: the current package declares the selected license cleanly, but a public repo should include the complete canonical license text.
