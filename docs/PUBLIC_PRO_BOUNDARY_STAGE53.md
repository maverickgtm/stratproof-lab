# Stage 53 — Public / Pro Boundary Final

## Decision

StratProof Lab should launch first as **Community Edition v0.1.0-community-preview**.

Community is the public AGPL edition. It is useful on its own, but it is intentionally limited to local, audit-only workflows.

Pro, Pro Plus, Team, and Enterprise are commercial roadmap editions. They can be delivered later as a private repo, hosted SaaS, licensed desktop/self-hosted package, or enterprise deployment.

## Community includes

- Local Workbench.
- Formula Builder basics.
- Main symbol selector and custom symbol input.
- Bybit and Binance public history download.
- Offline synthetic demo cache.
- Strict audit.
- Relaxed discovery audit.
- Strict vs relaxed Evidence Ladder.
- Visual evidence report.
- Markdown / JSON / threshold comparison reports.
- Saved ideas local preview.
- Pro feature preview cards.

## Community does not include

- Real payments.
- License server.
- Customer account system.
- Broker execution.
- Signal publishing workflows.
- Managed accounts.
- Paid feature unlocks.
- Wallet/private key handling.

## Pro roadmap

Planned Pro features:

- LONG + SHORT comparison.
- Batch formula audit.
- Strategy ranking.
- Advanced PDF reports.
- Expanded saved ideas library.
- Larger history limits.
- More connector workflows.

## Pro Plus roadmap

Planned Pro Plus features:

- Regime analysis.
- Session comparison.
- Multi-timeframe advanced reports.
- Portfolio-level evidence.
- Walk-forward style reviews.
- Advanced duplicate/leakage diagnostics.

## Team / Enterprise roadmap

Planned Team / Enterprise features:

- Team workspaces.
- Branded reports.
- Role-based access.
- Enterprise connector layer.
- Private deployment.
- Custom onboarding and support.

## Public messaging rule

Do not imply paid editions are already active. Public wording should say:

> Community is available now. Paid editions are planned / early access. No payment system is active in this repo.
