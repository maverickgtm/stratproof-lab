# Community vs Pro Boundary — Stage 45

## Community / GitHub

Purpose: prove the value of evidence-based strategy auditing.

Includes:

- Formula Builder basics
- Local Workbench
- Top symbols + custom symbols
- Bybit, Binance, OKX, Coinbase Exchange, and Kraken public history downloader
- Duplicate-safe cache and completed-candle evidence controls
- Offline demo cache
- Strict audit
- Relaxed discovery audit
- Strict vs Relaxed Evidence Ladder
- Visual Evidence Report
- Markdown/JSON/threshold reports
- Local saved ideas

Recommended limits:

- local single-user mode
- no cloud sync
- no team workspace
- no broker execution
- no managed accounts
- no guaranteed performance claims
- limited connectors active by default

## Pro

Purpose: daily research productivity.

Potential additions:

- larger history management
- richer indicator catalog
- batch formula comparison
- PDF/HTML premium exports
- saved idea versioning
- formula ranking
- broader provider importers

## Pro Plus

Purpose: advanced quant-style validation.

Potential additions:

- walk-forward validation
- regime-aware analysis
- session optimizer
- duplicate/leakage audit suite
- portfolio-level reports
- threshold optimizer
- multi-market data imports

## Team / Enterprise

Purpose: commercial evidence workflows.

Potential additions:

- users and roles
- shared workspaces
- branded reports
- private API
- white-label dashboard
- premium provider connectors
- deployment support
- compliance/audit logs
