# StratProof Lab v0.1.0 Community Preview

**Most trading strategies look good — until they are audited.**

StratProof Lab is an evidence engine for trading strategies. It helps traders, quants, educators, and teams turn ideas into audit hypotheses, test them against market data, inspect weak evidence, and generate reports before risking capital.

## What is included

- Formula Builder UI
- Evidence Report Builder UI
- Research Brain View
- Provider Connector Layer
- CSV Import Wizard
- Idea Lab Backtest Runner
- Multi-timeframe BTC Context Engine
- Indicator Block Library
- One-command public demo
- Multilingual foundation: EN / ES / PT / DE
- Public GitHub visual asset pack
- AGPL-3.0-or-later Community license

## Quick demo

```bash
pip install -r requirements.txt
python scripts/run_public_demo.py
```

Then open:

```text
reports/public_demo/DEMO_INDEX.md
```

## Release status

This is a **community preview**. It is intended for testing, education, feedback, and public validation of the product direction.

## Safety

StratProof Lab is audit-only by design. It generates evidence reports, not financial advice or guaranteed returns.
